# Getting Started

Welcome to AppForge! This guide will help you get up and running quickly.

## Installation

```bash
# Clone the repository
git clone https://github.com/sofn/AppForge.git

# Enter the project directory
cd AppForge
```

## Quick Start

After installation, you can start using AppForge right away:

```bash
# Run the application
./appforge run
```
